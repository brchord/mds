# Production Setup Guide & Toolkit

This guide outlines your professional audio production toolkit and the required installation order for Ableton Live Standard and Reason 14.

## 1. Installation Priority Order

1. **Ableton Live Standard Packs:** Download and install all content from your Ableton Browser to establish your baseline library.
2. **VST Suite:** Install Vital, Surge XT, TDR Nova, TDR Kotelnikov, Valhalla Supermassive, ChowTape, Lancaster Pulse 2, Voxengo SPAN, and Youlean Loudness Meter.
3. **Content Bundles:** Install UAD Explore (Free) and Komplete Start to populate your library.

---

## 2. Toolkit Summary

| **Category** | **Plugin / Component** | **Primary Purpose** |
| :--- | :--- | :--- |
| **Synthesizers** | Vital, Surge XT | Advanced wavetable & hybrid synthesis. |
| **Dynamics** | TDR Nova, TDR Kotelnikov | Surgical EQ & transparent compression. |
| **Creative FX** | Valhalla Supermassive | Expansive spatial effects. |
| **Saturation** | ChowTapeModel | Physical tape modeling. |
| **Cab Sim** | Lancaster Pulse 2 | Impulse response cabinet loader. |
| **Analysis** | Voxengo SPAN, Youlean Loudness | Spectral analysis & LUFS metering. |
| **Bundles** | UAD Explore, Komplete Start | Acoustic, percussion, & sound design libraries. |
| **Live Content** | Ableton Live Standard Packs | Core instruments, loops, & samples. |

---

## 3. Workflow Logic

- **Generation:** Use **Ableton Packs** for core sounds; **Vital/Surge/Komplete Start** for advanced design.
- **Processing:** Use **UAD Explore** for character; **TDR Nova/Kotelnikov** for precision; **ChowTape** for bus glue.
- **Monitoring:** Keep **Voxengo SPAN** and **Youlean Loudness** as your final gatekeepers for technical balance.
